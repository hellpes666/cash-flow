# Cash Flow

Cash Flow — это современный сервис управления личными финансами с упором на визуализацию денежного потока и удобный ежедневный учёт.
Проект помогает отслеживать доходы и расходы, видеть картину по месяцам и принимать более осознанные финансовые решения.

На первом этапе Cash Flow работает как веб‑приложение: backend на NestJS и frontend на Next.js, объединённые в одном репозитории.
Пользователь получает простой интерфейс для работы с транзакциями и дашбордами, а в дальнейшем планируется развитие AI‑ассистента и мобильного приложения.

## Основные идеи проекта

- Минималистичный и понятный интерфейс для повседневного учёта денег.
- Умные подсказки и аналитика, которые со временем смогут подстраиваться под привычки пользователя.
