# Cash Flow Frontend

Frontend Cash Flow — это клиентское приложение на Next.js + React + TypeScript, предоставляющее интерфейс для учёта доходов и расходов, просмотра дашбордов и работы с финансовыми целями.

## Архитектура и стек

- Next.js в качестве фреймворка: маршрутизация, рендеринг (SSR/SSG), при необходимости собственные API‑routes.  
- React + TypeScript для типобезопасных компонентов и удобной разработки.  
- Вёрстка и стилизация (Tailwind CSS или выбранная UI‑библиотека, если используется в проекте) для быстрой сборки интерфейса и адаптивности.  
- Взаимодействие с backend через REST API (NestJS): получение транзакций, категорий, аккаунтов и агрегированных данных для графиков.

## Структура интерфейса

- Главный дашборд: обзор текущего баланса, доходов и расходов за период, ключевые метрики по cash flow.  
- Страница транзакций: список операций с фильтрами по дате, категории и типу (доход/расход), формы создания и редактирования.  
- Страница категорий и счетов: управление категориями и счетами, используемыми в учёте.  
- В перспективе — страница целей и расширенной аналитики с графиками и подсказками.

## Паттерны и практики

- Разделение на страницы и презентационные компоненты: страницы отвечают за данные и навигацию, UI‑компоненты — за отображение.  
- Хуки для работы с данными (например, `useTransactions`, `useAccounts`, `useDashboard`), инкапсулирующие запросы к API и состояния загрузки/ошибок.  
- Глобальное состояние (Context или стейт‑менеджер, если будет добавлен) для общих сущностей: текущий пользователь, выбранный период, фильтры.  
- Унифицированные компоненты UI (кнопки, инпуты, таблицы, карточки), переиспользуемые на разных страницах для консистентного интерфейса.

## Работа с визуализацией

- Графики для отображения движения денег по времени (cash inflow/outflow, чистый денежный поток).  
- Диаграммы по категориям расходов и доходов, чтобы быстро видеть основные направления трат и источники дохода.  
- Адаптивный дизайн: дашборды и таблицы оптимизированы под разные размеры экранов для удобной работы и на десктопе, и на мобильных устройствах.
